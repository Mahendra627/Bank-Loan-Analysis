# 🏦 Bank Loan Analysis

## 📌 Project Overview
This is an end-to-end Data Analytics project that analyzes bank loan data to evaluate loan performance, customer behavior, repayment trends, and key business metrics. The project was completed using SQL, Python, Excel, and Power BI to transform raw data into meaningful business insights.

---

## 🎯 Project Objectives
- Analyze overall loan performance.
- Identify Good Loans and Bad Loans.
- Evaluate loan applications and funding trends.
- Analyze customer demographics and loan purposes.
- Build interactive dashboards for business decision-making.

---

## 💡 Key Insights
- Good Loans accounted for approximately **86.2%** of total loans.
- Total Funded Amount exceeded **$435 Million**.
- Total Amount Received exceeded **$473 Million**.
- Most loan applications came from customers with stable employment.
- Debt-to-Income ratio and interest rate significantly impacted loan performance.
